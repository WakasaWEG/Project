Практика ПМ.03

Это описание для работы с проектом.
Для работы спроектом потребуется:

Приложения:
ЯндексБраузер(или любой другой);
VisualStidioCode;
XAMPP.

Что бы запустить проект необходимо воспользоваться таблицами из БД(Файл приложен)
Их можно экспортировать на сайте phpMyAdmin.

Затем необходимо запустить XAMPP, после перейти по ссылке: http://localhost/Project/Authorization.php
Отсюда можно начинать работу:
Можно авторизоваться или зарегистрироваться.
